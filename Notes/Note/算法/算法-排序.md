### 冒泡排序
以从小到大的排列数组为例，依次比较相邻两个元素的大小，若前一个数字大于后一个数字，则替换两者位置。不断重复直指排列完毕

```javascript
// javascript的实现
((list) => {
  let length = list.length;
  
  const Sequence = (length) => {
    for(let i = 0; i < length - 1; i++) {
      if(list[i] > list[i + 1]) {
        let cache = list[i + 1];
        list[i + 1] = list[i];
        list[i] = cache;
      }
      if(i == length - 2) {
        Sequence(length - 1);
      }
    }
  }
  
  Sequence(length);
  console.log(list); // [0, 1, 1, 2, 3, 4, 5, 7, 11]
})([3, 5, 1, 7, 0, 11, 2, 4, 1]);
```
